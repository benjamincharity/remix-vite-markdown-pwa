## Text

### Headings

Use `#` for a level 1 heading, `##` for level 2, and so on up to `######` for
level 6.

### Emphasis

- **Bold** with `**` or `__`.
- _Italic_ with `*` or `_`.
- ~~Strikethrough~~ with `~~`.

## Code

### Inline Code

Use single backticks: `var example = true;`

This is an array `[1, 2, 3]{:js}` of numbers 1 through 3.

### Code Block

```javascript title="Basic syntax highlighting"
function test() {
  console.log('Standard output');
}
```

### Line Highlighting

```js {2} title="Highlight single line"
function test() {
  console.log('notice I am highlighted?');
}
```

```js {2,3} title="Highlight multiple lines"
function test() {
  console.log('notice I am highlighted?');
  console.log('and, so am I!');
}
```

```js {2,4-7} title="Highlight range"
function test() {
  console.log('one');
  console.log('two');
  console.log('three');
  console.log('four');
  console.log('five');
  console.log('six');
}
```

### Highlight word

```js "test" title="Highlight word"
function test() {
  console.log('notice that test is highlighted?');
}
```

```ts title="Highlight word by comment"
// [!code word:options:2]
const options = { foo: 'bar' };
options.foo = 'baz';
console.log(options.foo); // this one will not be highlighted
```

### Highlight by IDs

```js /age/#a /name/#a /email/#a /setAge/#b /setName/#c /50/#d /'Taylor'/#e /setEmail/#f /example.com/#g title="Highlight multiple by ID"
const [age, setAge] = useState(50);
const [name, setName] = useState('Taylor');
const [email, setEmail] = useState('taylor@example.com');
```

```ts /ab/#a title="Highlight by ID"
const ab = 12;
const abcd = 1234;
```

### Diff

```ts
export function foo() {
  console.log('hewwo'); // [!code --]
  console.log('hello'); // [!code ++]
}
```

```ts
export function foo() {
  -console.log('hewwo');
  +console.log('hello');
}
```
